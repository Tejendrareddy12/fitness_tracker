# 🏋️ Fitness Tracker (Python Project)

The Fitness Tracker is a simple console-based Python application designed to help users track their daily physical activities and monitor calories burned.

## 🚀 Features
- Add fitness activities (Running, Walking, etc.)
- Record duration and calories burned
- View all activity history
- Calculate total calories burned
- Simple and user-friendly interface

## 🛠️ Technologies Used
- Python
- Basic Data Structures (List, Dictionary)
## 📌 Project Purpose
This project is built for learning Python fundamentals such as functions, loops, and data handling. It is suitable for beginners and college-level mini projects.

## 🔮 Future Enhancements
- Save data to file (CSV/JSON)
- Add Graphs using Matplotlib
- Build GUI using Tkinter
- Add user login system
